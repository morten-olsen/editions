# Quiet by design

<div style={{ fontSize: "1.44rem", color: "var(--color-ink-secondary)", lineHeight: 1.625, marginTop: "1.5rem", marginBottom: "4rem" }}>
  A design system built around a single conviction: the best interface is the one that disappears. Every token, every component, every interaction in Editions serves one purpose — to let you read, think, and feel done.
</div>

<div style={{ height: "1px", background: "var(--color-border)", margin: "4rem 0" }} />

<div style={{ display: "grid", gridTemplateColumns: "repeat(3, 1fr)", gap: "3rem", margin: "4rem 0" }}>
  <div>
    <div style={{ fontSize: "2.074rem", fontFamily: "var(--font-serif)", lineHeight: 1, marginBottom: "0.75rem" }}>Calm</div>
    <div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", lineHeight: 1.625 }}>
      No unread counts. No urgency signals. No red badges screaming for attention. The interface breathes so the reader can too.
    </div>
  </div>
  <div>
    <div style={{ fontSize: "2.074rem", fontFamily: "var(--font-serif)", lineHeight: 1, marginBottom: "0.75rem" }}>Finite</div>
    <div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", lineHeight: 1.625 }}>
      Every surface has a natural end. "I am done" is a first-class feature — not an impossibility hidden behind infinite scroll.
    </div>
  </div>
  <div>
    <div style={{ fontSize: "2.074rem", fontFamily: "var(--font-serif)", lineHeight: 1, marginBottom: "0.75rem" }}>Focused</div>
    <div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", lineHeight: 1.625 }}>
      When you're reading, you're reading. The interface steps back. Content is the foreground; experience is the language.
    </div>
  </div>
</div>

## The feeling

<div style={{ color: "var(--color-ink-secondary)", lineHeight: 1.625, marginBottom: "2rem" }}>
  Reading an Edition should feel like settling into a favourite chair with a good magazine — quiet, deliberate, finite. The technology available today should let us build something meaningfully better than both firehose timelines and algorithmic feeds.
</div>

<div style={{ color: "var(--color-ink-secondary)", lineHeight: 1.625, marginBottom: "2rem" }}>
  This design system encodes that feeling into reusable decisions. Warm surfaces that feel like paper. Ink that's gentle on the eyes. Typography that invites long reading. Spacing that gives every element room to breathe.
</div>

## Visual language

<div style={{ display: "grid", gridTemplateColumns: "repeat(2, 1fr)", gap: "2.5rem", margin: "2.5rem 0" }}>
  <div>
    <div style={{ fontSize: "0.833rem", letterSpacing: "0.04em", textTransform: "uppercase", color: "var(--color-ink-tertiary)", marginBottom: "0.75rem" }}>Typography</div>
    <div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", lineHeight: 1.625 }}>
      A serif for reading, a sans-serif for interface. The type scale follows a minor third ratio — gentle, never jarring. Generous line heights invite the eye to linger.
    </div>
  </div>
  <div>
    <div style={{ fontSize: "0.833rem", letterSpacing: "0.04em", textTransform: "uppercase", color: "var(--color-ink-tertiary)", marginBottom: "0.75rem" }}>Color</div>
    <div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", lineHeight: 1.625 }}>
      Warm surfaces like quality paper. Ink that's charcoal, not black. A single muted accent — sage green — that speaks with calm authority.
    </div>
  </div>
  <div>
    <div style={{ fontSize: "0.833rem", letterSpacing: "0.04em", textTransform: "uppercase", color: "var(--color-ink-tertiary)", marginBottom: "0.75rem" }}>Space</div>
    <div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", lineHeight: 1.625 }}>
      Generous. White space isn't empty — it's where the reading happens. Every element has room to breathe. Density is the enemy of calm.
    </div>
  </div>
  <div>
    <div style={{ fontSize: "0.833rem", letterSpacing: "0.04em", textTransform: "uppercase", color: "var(--color-ink-tertiary)", marginBottom: "0.75rem" }}>Motion</div>
    <div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", lineHeight: 1.625 }}>
      Gentle, never abrupt. Transitions feel like turning a page — smooth, purposeful, invisible when done right.
    </div>
  </div>
</div>
