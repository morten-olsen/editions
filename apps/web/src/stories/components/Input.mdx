## Anatomy

<div style={{ display: "flex", flexDirection: "column", gap: "0.75rem", fontSize: "0.833rem", color: "var(--color-ink-secondary)", lineHeight: 1.625, maxWidth: "65ch" }}>
  <div><strong style={{ color: "var(--color-ink)" }}>Label</strong> — Always visible when the field has meaning beyond its placeholder. Set via the <code style={{ fontSize: "0.75rem", fontFamily: "var(--font-mono)", background: "var(--color-surface-sunken)", padding: "0.125rem 0.375rem", borderRadius: "0.25rem" }}>label</code> prop.</div>
  <div><strong style={{ color: "var(--color-ink)" }}>Description</strong> — Supplementary guidance below the label. Keep it to one short sentence.</div>
  <div><strong style={{ color: "var(--color-ink)" }}>Error</strong> — Replaces the description when validation fails. Uses the critical color but remains understated.</div>
  <div><strong style={{ color: "var(--color-ink)" }}>Placeholder</strong> — Shown when the field is empty. Use for format hints, not labels.</div>
</div>
