# Select

<div style={{ fontSize: "1.2rem", color: "var(--color-ink-secondary)", lineHeight: 1.625, marginTop: "1rem", marginBottom: "4rem", maxWidth: "65ch" }}>
  A dropdown for choosing one value from a predefined list. Used for edition lookback windows, schedule frequencies, and sort orders. The trigger matches the input styling for visual consistency in forms.
</div>

<div style={{ height: "1px", background: "var(--color-border)", marginBottom: "4rem" }} />

## Anatomy

<div style={{ display: "flex", flexDirection: "column", gap: "0.75rem", fontSize: "0.833rem", color: "var(--color-ink-secondary)", lineHeight: 1.625, maxWidth: "65ch" }}>
  <div><strong style={{ color: "var(--color-ink)" }}>Root</strong> — Manages state. Accepts <code style={{ fontSize: "0.75rem", fontFamily: "var(--font-mono)", background: "var(--color-surface-sunken)", padding: "0.125rem 0.375rem", borderRadius: "0.25rem" }}>defaultValue</code> for uncontrolled or <code style={{ fontSize: "0.75rem", fontFamily: "var(--font-mono)", background: "var(--color-surface-sunken)", padding: "0.125rem 0.375rem", borderRadius: "0.25rem" }}>value</code> + <code style={{ fontSize: "0.75rem", fontFamily: "var(--font-mono)", background: "var(--color-surface-sunken)", padding: "0.125rem 0.375rem", borderRadius: "0.25rem" }}>onValueChange</code> for controlled.</div>
  <div><strong style={{ color: "var(--color-ink)" }}>Trigger</strong> — Displays the selected value with a chevron indicator. Shows "Select..." as placeholder when empty.</div>
  <div><strong style={{ color: "var(--color-ink)" }}>Content</strong> — The dropdown panel. Positioned automatically, transitions with scale and opacity.</div>
  <div><strong style={{ color: "var(--color-ink)" }}>Item</strong> — Individual option. Shows a check mark when selected. Highlighted on keyboard navigation.</div>
  <div><strong style={{ color: "var(--color-ink)" }}>Group + GroupLabel</strong> — Optional grouping for longer lists. The label renders as a small uppercase header.</div>
</div>

<div style={{ height: "1px", background: "var(--color-border)", margin: "4rem 0" }} />

## Guidelines

<div style={{ display: "grid", gridTemplateColumns: "1fr 1fr", gap: "2.5rem" }}>
  <div>
    <div style={{ fontSize: "0.833rem", letterSpacing: "0.04em", textTransform: "uppercase", color: "var(--color-ink-tertiary)", marginBottom: "0.75rem" }}>When to use</div>
    <div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", lineHeight: 1.625 }}>
      5 or more options where only one can be selected. For fewer options, consider a radio group or segmented control.
    </div>
  </div>
  <div>
    <div style={{ fontSize: "0.833rem", letterSpacing: "0.04em", textTransform: "uppercase", color: "var(--color-ink-tertiary)", marginBottom: "0.75rem" }}>Keep lists short</div>
    <div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", lineHeight: 1.625 }}>
      Dropdowns hide their options until opened. If the user needs to compare options, consider showing them all at once with radio buttons.
    </div>
  </div>
</div>

</div>
