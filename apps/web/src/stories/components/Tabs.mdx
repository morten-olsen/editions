# Tabs

<div style={{ fontSize: "1.2rem", color: "var(--color-ink-secondary)", lineHeight: 1.625, marginTop: "1rem", marginBottom: "4rem", maxWidth: "65ch" }}>
  Organize related content into panels the reader can switch between without leaving the page. The active tab is indicated by a quiet accent underline — no filled backgrounds, no heavy borders. Inactive tabs are tertiary ink, stepping back to let the content panel speak.
</div>

<div style={{ height: "1px", background: "var(--color-border)", marginBottom: "4rem" }} />

## Anatomy

<div style={{ display: "flex", flexDirection: "column", gap: "0.75rem", fontSize: "0.833rem", color: "var(--color-ink-secondary)", lineHeight: 1.625, maxWidth: "65ch" }}>
  <div><strong style={{ color: "var(--color-ink)" }}>Root</strong> — Container that manages which tab is active. Set <code style={{ fontSize: "0.75rem", fontFamily: "var(--font-mono)", background: "var(--color-surface-sunken)", padding: "0.125rem 0.375rem", borderRadius: "0.25rem" }}>defaultValue</code> for the initially active panel.</div>
  <div><strong style={{ color: "var(--color-ink)" }}>List</strong> — The tab bar. A row of tabs separated by a bottom border.</div>
  <div><strong style={{ color: "var(--color-ink)" }}>Tab</strong> — Individual tab trigger. Requires a <code style={{ fontSize: "0.75rem", fontFamily: "var(--font-mono)", background: "var(--color-surface-sunken)", padding: "0.125rem 0.375rem", borderRadius: "0.25rem" }}>value</code> that matches its corresponding Panel.</div>
  <div><strong style={{ color: "var(--color-ink)" }}>Panel</strong> — Content area revealed when its tab is active.</div>
</div>

## Guidelines

<div style={{ display: "grid", gridTemplateColumns: "1fr 1fr", gap: "2.5rem" }}>
  <div>
    <div style={{ fontSize: "0.833rem", letterSpacing: "0.04em", textTransform: "uppercase", color: "var(--color-ink-tertiary)", marginBottom: "0.75rem" }}>Keep tabs few</div>
    <div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", lineHeight: 1.625 }}>
      Two to four tabs. More than that overwhelms the bar and hides content behind too many doors. Consider a sidebar or page navigation instead.
    </div>
  </div>
  <div>
    <div style={{ fontSize: "0.833rem", letterSpacing: "0.04em", textTransform: "uppercase", color: "var(--color-ink-tertiary)", marginBottom: "0.75rem" }}>Label clearly</div>
    <div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", lineHeight: 1.625 }}>
      One or two words per tab. The label should make the content immediately predictable — no surprises behind the click.
    </div>
  </div>
</div>

</div>
