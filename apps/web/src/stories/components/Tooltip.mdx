# Tooltip

<div style={{ fontSize: "1.2rem", color: "var(--color-ink-secondary)", lineHeight: 1.625, marginTop: "1rem", marginBottom: "4rem", maxWidth: "65ch" }}>
  A small, non-interactive label that appears on hover or focus. Used for supplementary context — icon button labels, truncated text, or brief explanations. Tooltips should never contain essential information.
</div>

<div style={{ height: "1px", background: "var(--color-border)", marginBottom: "4rem" }} />

## Props

<div style={{ display: "flex", flexDirection: "column", gap: "0.75rem", fontSize: "0.833rem", color: "var(--color-ink-secondary)", lineHeight: 1.625, maxWidth: "65ch" }}>
  <div><strong style={{ color: "var(--color-ink)" }}>content</strong> — The text or node displayed in the tooltip.</div>
  <div><strong style={{ color: "var(--color-ink)" }}>children</strong> — The trigger element. Must accept a ref (use a native element or forwardRef component).</div>
  <div><strong style={{ color: "var(--color-ink)" }}>side</strong> — Preferred placement: top (default), bottom, left, or right. Automatically repositions if there isn't enough space.</div>
</div>

<div style={{ height: "1px", background: "var(--color-border)", margin: "4rem 0" }} />

## Provider

<div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", lineHeight: 1.625, maxWidth: "65ch" }}>
  Wrap your app (or a subtree) in <code style={{ fontSize: "0.75rem", fontFamily: "var(--font-mono)", background: "var(--color-surface-sunken)", padding: "0.125rem 0.375rem", borderRadius: "0.25rem" }}>TooltipProvider</code> to coordinate delay timing across tooltips. Once one tooltip has been shown, subsequent tooltips in the group appear immediately — preventing a sluggish feel when scanning a toolbar.
</div>

<div style={{ height: "1px", background: "var(--color-border)", margin: "4rem 0" }} />

## Guidelines

<div style={{ display: "grid", gridTemplateColumns: "1fr 1fr", gap: "2.5rem" }}>
  <div>
    <div style={{ fontSize: "0.833rem", letterSpacing: "0.04em", textTransform: "uppercase", color: "var(--color-ink-tertiary)", marginBottom: "0.75rem" }}>Keep it short</div>
    <div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", lineHeight: 1.625 }}>
      A few words, not a sentence. If you need more space, use a Popover instead.
    </div>
  </div>
  <div>
    <div style={{ fontSize: "0.833rem", letterSpacing: "0.04em", textTransform: "uppercase", color: "var(--color-ink-tertiary)", marginBottom: "0.75rem" }}>Not for essential info</div>
    <div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", lineHeight: 1.625 }}>
      Tooltips are invisible on touch devices. Any information critical to using the interface should be visible without hovering.
    </div>
  </div>
</div>
