# Article

<div style={{ fontSize: "1.2rem", color: "var(--color-ink-secondary)", lineHeight: 1.625, marginTop: "1rem", marginBottom: "4rem", maxWidth: "65ch" }}>
  A composable article rendering system inspired by Radix UI's compound component pattern. Instead of a monolithic article component, the system provides atomic parts that compose into style presets and context-specific footers. One article, many presentations.
</div>

<div style={{ height: "1px", background: "var(--color-border)", marginBottom: "4rem" }} />

## Compound parts

Each part takes explicit props including <code style={{ fontSize: "0.75rem", background: "var(--color-surface-sunken)", padding: "0.15rem 0.35rem", borderRadius: "0.25rem" }}>delay</code> for staggered entrance animations.

## Style selector

<div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", marginBottom: "2rem", maxWidth: "65ch" }}>
  The <code style={{ fontSize: "0.75rem", background: "var(--color-surface-sunken)", padding: "0.15rem 0.35rem", borderRadius: "0.25rem" }}>ArticleView</code> component accepts a <code style={{ fontSize: "0.75rem", background: "var(--color-surface-sunken)", padding: "0.15rem 0.35rem", borderRadius: "0.25rem" }}>style</code> prop to select which preset renders the article. This enables a single component to serve the article page, magazine spreads, and any future presentation.
</div>

```tsx
import { ArticleView } from 'components/article/article.presets.tsx';

<ArticleView
  style="editorial"      // "editorial" | "hero" | "compact"
  title={article.title}
  sourceName={article.sourceName}
  content={article.content}
  footer={<FeedFooter vote={vote} onVote={handleVote} />}
/>
```

## Podcasts and the children slot

<div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", marginBottom: "2rem", maxWidth: "65ch" }}>
  Every preset accepts a <code style={{ fontSize: "0.75rem", background: "var(--color-surface-sunken)", padding: "0.15rem 0.35rem", borderRadius: "0.25rem" }}>children</code> prop that renders between the byline and body. This is the natural injection point for a media player, an embed, or any content that sits between the article header and its body text.
</div>

```tsx
import { Editorial } from 'components/article/article.presets.tsx';
import { MediaPlayer } from 'components/media-player.tsx';

<Editorial
  title={podcast.title}
  sourceName={podcast.sourceName}
  summary={podcast.summary}
  imageUrl={podcast.imageUrl}
  content={podcast.showNotes}
  sourceType="podcast"
  footer={<FeedFooter vote={vote} onVote={handleVote} onDone={goBack} />}
>
  <div className="mt-8">
    <MediaPlayer
      mediaUrl={podcast.mediaUrl}
      mediaType="audio/mpeg"
    />
  </div>
</Editorial>
```

<div style={{ fontSize: "0.833rem", color: "var(--color-ink-secondary)", lineHeight: 1.625, marginBottom: "2rem", maxWidth: "65ch" }}>
  The same pattern works with any preset — Hero, Compact, or a fully custom compound composition. Podcast articles in the magazine view use a dedicated <code style={{ fontSize: "0.75rem", background: "var(--color-surface-sunken)", padding: "0.15rem 0.35rem", borderRadius: "0.25rem" }}>PodcastLayout</code> that composes Article parts directly for a more tailored presentation (centered album art, waveform decoration, etc.).
</div>

## Composing a custom layout

<div style={{ color: "var(--color-ink-secondary)", fontSize: "0.833rem", marginBottom: "2rem", maxWidth: "65ch" }}>
  When presets don't fit, compose directly with the compound parts. This example creates a side-by-side layout with the summary prominently displayed:
</div>

```tsx
import * as Article from 'components/article/article.tsx';
import { FeedFooter } from 'components/article/article.footers.tsx';

<Article.Root className="max-w-wide mx-auto grid lg:grid-cols-2 gap-16">
  <div>
    <Article.Source name={sourceName} delay={0} />
    <Article.Title size="xl" delay={0.1}>{title}</Article.Title>
    <Article.Summary delay={0.2}>{summary}</Article.Summary>
    <Article.Byline author={author} publishedAt={publishedAt} delay={0.3} />
  </div>
  <div>
    <Article.Image src={imageUrl} aspect="photo" delay={0.15} />
  </div>
  <div className="lg:col-span-2">
    <Article.Body content={content} delay={0.4} />
    <FeedFooter vote={vote} onVote={handleVote} onDone={goBack} />
  </div>
</Article.Root>
```
