# Discovery

<div style={{ fontSize: "1.2rem", color: "var(--color-ink-secondary)", lineHeight: 1.625, marginTop: "1rem", marginBottom: "4rem", maxWidth: "65ch" }}>

    <p style={{ margin: "0 0 0.75rem" }}>The page uses three tabs — **Editions** (the high-value action), **Focuses**, and **Sources** — ordered by impact.     <p style={{ margin: 0 }}>Items already in the user's account show a quiet "Added" label with a checkmark instead of the add button. This provides immediate feedback without being noisy.</p>

## Cascading adoption

<div style={{ display: "flex", gap: "4rem", marginBottom: "4rem", alignItems: "flex-start" }}>
  <div style={{ width: "20rem", flexShrink: 0 }}>
    <div style={{ border: "1px solid var(--color-border)", borderRadius: "0.5rem", padding: "0.75rem", background: "var(--color-surface)", fontSize: "0.7rem", fontFamily: "var(--font-mono)", display: "flex", flexDirection: "column", gap: "0.375rem" }}>
      <div style={{ color: "var(--color-ink)", fontWeight: 500 }}>Adopt "Morning Briefing"</div>
      <div style={{ color: "var(--color-ink-tertiary)", paddingLeft: "0.75rem" }}>├─ Focus: World News</div>
      <div style={{ color: "var(--color-ink-faint)", paddingLeft: "1.5rem" }}>├─ BBC News (new)</div>
      <div style={{ color: "var(--color-ink-faint)", paddingLeft: "1.5rem" }}>├─ Reuters (new)</div>
      <div style={{ color: "var(--color-accent)", paddingLeft: "1.5rem" }}>├─ The Guardian (exists)</div>
      <div style={{ color: "var(--color-ink-faint)", paddingLeft: "1.5rem" }}>└─ AP News (new)</div>
      <div style={{ color: "var(--color-ink-tertiary)", paddingLeft: "0.75rem" }}>├─ Focus: Technology</div>
      <div style={{ color: "var(--color-ink-faint)", paddingLeft: "1.5rem" }}>├─ Ars Technica (new)</div>
      <div style={{ color: "var(--color-ink-faint)", paddingLeft: "1.5rem" }}>└─ ...</div>
      <div style={{ color: "var(--color-ink-tertiary)", paddingLeft: "0.75rem" }}>└─ Focus: Science</div>
      <div style={{ color: "var(--color-ink-faint)", paddingLeft: "1.5rem" }}>└─ ...</div>
    </div>
  </div>
  <div style={{ fontSize: "0.833rem", color: "var(--color-ink-secondary)", lineHeight: 1.625 }}>
    <p style={{ margin: "0 0 0.75rem" }}>When a user adopts an edition config, the system walks the dependency tree: for each focus the edition references, and for each source each focus references.</p>
    <p style={{ margin: "0 0 0.75rem" }}>At each level, deduplication kicks in: sources match by **URL** (a natural identity), focuses and edition configs match by **origin_id** (linking back to the catalog template they came from).</p>
    <p style={{ margin: 0 }}>The result: a user can adopt multiple edition configs that share focuses without ending up with duplicate topic feeds. Two configs that both reference "Technology" will share the same user focus.</p>
  </div>
</div>
