## Don't repeat articles across editions

<div style={{ display: "flex", gap: "4rem", marginBottom: "4rem", alignItems: "flex-start" }}>
  <div style={{ width: "20rem", flexShrink: 0 }}>
    <Checkbox
      label="Don't repeat articles across editions"
      description="Articles that appeared in a previous issue of this digest won't be included again"
    />
  </div>
  <div style={{ fontSize: "0.833rem", color: "var(--color-ink-secondary)", lineHeight: 1.625 }}>
    When on, the edition remembers which articles it has already surfaced in past issues and won't include them again. This is about <em>between</em> editions — articles that appeared in yesterday's briefing won't reappear in today's. Good for daily editions where you want each issue to feel genuinely new. Turn it off for infrequent digests where some overlap is acceptable.
  </div>
</div>
