# Magazine view

<div style={{ fontSize: "1.2rem", color: "var(--color-ink-secondary)", lineHeight: 1.625, marginTop: "1rem", marginBottom: "4rem", maxWidth: "65ch" }}>
  The magazine view is the primary reading experience for editions. Each page takes the full viewport — there is no infinite scroll, no "load more". Reading an edition is like reading an issue of a magazine: it has a cover, a table of contents, sections, and an ending. Being done is the feature.
</div>
